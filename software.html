SuSiEx

SuSiEx is a cross-ancestry fine-mapping tool built on the SuSiE framework. It integrates GWAS summary statistics with ancestry-matched LD reference panels, models shared and population-specific genetic architecture, and prioritizes likely causal variants through credible sets and posterior inclusion probabilities.

Paper GitHub

ArchaicSeeker2.0

ArchaicSeeker2.0 identifies introgressed sequences from known or unknown archaic hominins and reconstructs multiple-wave introgression histories. The method was designed to refine models of Neanderthal-like and Denisovan-like ancestry, helping connect archaic introgression with modern human diversity and adaptation.

Paper GitHub

MultiWaver2.0

MultiWaver2.0 reconstructs complex admixture histories using the length distribution of ancestry tracts. Compared with the original discrete-wave model, it supports both discrete and continuous gene flow and can select among candidate admixture models to estimate timing and contribution of ancestral sources.

Paper GitHub

MultiWaver

MultiWaver is the original framework for inferring multiple-wave admixture from ancestry-track length distributions under a general discrete admixture model. It estimates the number of admixture waves and their parameters, providing a practical way to translate local ancestry patterns into population-history models.

Paper GitHub
